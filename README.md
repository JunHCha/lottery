### LOTTERY
#### - LOTTERY란?
여기에 LOTTERY 설명을 써주세요.
#### - LOTTERY를 만들게 된 계기
실제로 크리스피 크림 도넛에서 알바를 하면서 발주량과 실제 판매량이 많은 요인들(e.g. 날씨, 주변 행사 등)로 인해 제각각으로 변하며 
남는 재고 처리에 애를 쓴 경험을 토대로 LOTTERY란 서비스를 만들게 되었습니다.
